POW003_String_index
Program Description:
	Write a program that demostartes the use of the String class and
	how to manipulate its characters by calling the index position.
	
Algorithm:
1) Ask the user to enter a string.
2) Display the string entered by the user.
3) Display all of the index number and its character.
4) Ask the user to enter an index number.
5) Display the character at the given index.
6) Ask the user to Enter his/her name.
7) Create a new string and store the name of the user in reverse order.
8) Display the name and its reverse order.
9) Display end of the program.

HINT:	.charAt()  .length()
